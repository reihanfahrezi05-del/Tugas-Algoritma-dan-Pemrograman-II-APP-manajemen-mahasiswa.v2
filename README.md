# 🎓 Manajemen Data Mahasiswa v2 (Gamified Edition)

Projek manajemen mahasiswa menggunakan Streamlit dengan fitur OOP (Object-Oriented Programming), enkapsulasi yang benar, validasi input yang ketat, dan visualisasi interaktif bertema gamifikasi ala Duolingo.

## Fitur Utama

- **🌟 Rekrut Siswa Baru:** Menambahkan mahasiswa dengan validasi regex NIM (harus 5 digit angka), validasi duplikasi NIM, dan nama kosong.
- **📖 Buku Daftar Siswa:** Menampilkan seluruh data mahasiswa dengan layout rapi dan informatif.
- **🔍 Lacak Siswa:** Pencarian linear search berdasarkan NIM.
- **⚙️ Edit & Hapus:** Fitur CRUD lengkap (Update & Delete).
- **🏆 Ranking:** Mengurutkan data berdasarkan NIM dengan algoritma Bubble Sort.

## Cara Menjalankan

1. Install dependencies:
   ```bash
   pip install streamlit
   ```
2. Jalankan aplikasi:
   ```bash
   streamlit run app.py
   ```
